R√©gression lin√©aire avec NumPy

La r√©gression lin√©aire est une technique statistique utilis√©e pour mod√©liser la relation entre une variable d√©pendante (souvent not√©e y) et une ou plusieurs variables ind√©pendantes (not√©es x). En termes simples, c’est une m√©thode qui permet de tracer une ligne droite (appel√©e droite de r√©gression) qui s’ajuste le mieux aux donn√©es.

Calcul de la R√©gression Lin√©aire (√† la main)

  1. Concept de base: La formule de la r√©gression lin√©aire simple est : [ y = ax + b ] o√π :

    • y est la variable d√©pendante.
    • x est la variable ind√©pendante.
    • a est la pente de la ligne.
    • b est l’ordonn√©e √† l’origine (le point o√π la ligne coupe l’axe des y).

  2. Calcul de a et b: Pour trouver les valeurs de a et b, on utilise les formules suivantes : [ a = \frac{n(\sum xy) - (\sum x)(\sum y)}{n(\sum x^2) - (\sum x)^2} ] [ b = \frac{(\sum y)(\sum x^2) - (\sum x)(\sum xy)}{n(\sum x^2) - (\sum x)^2} ] o√π (n) est le nombre de points de donn√©es.

  3. Exemple en Python:

# Calcul des param√®tres

sum_xy = sum([x[i]*y[i] for i in range(n)])

a = (n * sum_xy - sum_x * sum_y) / (n * sum_x_sq - sum_x**2)
b = (sum_y * sum_x_sq - sum_x * sum_xy) / (n * sum_x_sq - sum_x**2)

print(f'Pente (a): {a}')
print(f'Ordonn√©e √† l\'origine (b): {b}')

Calcul de la R√©gression Lin√©aire avec NumPy

NumPy simplifie grandement le processus de calcul des param√®tres de la r√©gression lin√©aire. Voici comment :
